# BrowserMusic - File Definitions

## .pl file
a .pl file stores song and artwork data for an individual playlist

```json
{
	"artwork": "base64 encoded string",
	"songs": [{
		"Title": "",
		"Artist": "",
		"Album": "",
		"Artwork": "",
		"Path": ""
	}]
}
```

## .theme file
a .theme contains application themes, with the properties below

```json
{
	"Primary1": "",
	"Primary2": "",
	"Primary3": "",
	"Secondary1": "",
	"Secondary2": "",
	"Secondary3": "",
	"Background1": "",
	"Background2": ""
}
```
